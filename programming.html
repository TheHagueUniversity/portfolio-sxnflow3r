PROCESSING

Visual Art w/Processing

Programming

Personal Project

Challenge: Interactive Patterns

Generate an interactive image that incorporates several dynamic patterns. Incorporate loops to create patterns and create functions for reusable code. The interactive image should react to user actions. Express yourself creatively and make it beautiful or interesting or fun

Outcome:

Interactive art piece revolving constant looping shapes, involving the mouse tool.

This piece of processing was created as a sketching tool using key press functions. The artwork is of numerous shapes that start from small to big. Key 1 is for circles (looks like a hurricane),Key 2 is for squares, and Key 3 is for triangles. You have to keep pressing on the key to get the circles continuously getting bigger. Then, I wanted to add some colour to the artwork. If you press g, it will give a change in stroke colour, and if you press r, it will give a change in fill colour. When I was creating this, I was thinking of what to do on the spot. I know I wanted to create motion in my artwork that is interactive. I am really happy with the outcome. It is not only satisfying to play around with but to watch the circles and rectangles get bigger and create dimensions is really fun
